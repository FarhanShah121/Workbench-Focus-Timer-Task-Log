    What is the Pomodoro Technique?

    The Pomodoro Technique is a time-management method built around short, focused bursts of work separated by deliberate rest. You work with full attention for 25 minutes on a single task, then step away for a 5-minute break. Each 25-minute burst is called a "pomodoro." The rhythm exists to protect your attention: knowing a break is coming makes it easier to resist distraction right now, and the short breaks keep your focus from decaying over a long session.

    The rhythm this app follows

    • Work with focus for 25 minutes.
    • When the timer ends, the app automatically switches to a 5-minute break.
    • When the break ends, the app switches back to a fresh 25-minute focus session.
    • Each completed focus session adds one mark to today's tally.

    How to use Workbench

    1. Open the Timer tab. The dial shows 25:00 and is labeled "Focus session."
    2. Press Start to begin counting down. The brass ring drains clockwise as time passes.
    3. Press Start again (it becomes Pause) to pause, and press it once more to resume from where you left off.
    4. Reset stops the countdown and returns the current session to its full starting time.
    5. When a focus session finishes, the screen flashes, a chime plays, one tally mark is added, and a 5-minute break begins automatically.
    6. When the break finishes, the screen flashes and chimes again, and a new 25-minute focus session begins.
    7. Use the Task Ledger panel any time — type a task and press Add or hit Enter. Check the box to mark it done, and use the ✕ button to delete it.
    8. Your tasks and today's tally are saved automatically in this browser, so refreshing or closing the page won't lose them.

    Notes

    • The tally resets each new calendar day, so it always reflects "today."
    • Sound requires one interaction with the page first (a browser rule, not a bug) — pressing Start satisfies this.
    • All data lives only in this browser's local storage. Clearing your browser data will clear it too.
